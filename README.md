## 本程序是Windows专用的，不支持其他系统
## 需要用户手动将node.exe和ffmpeg.exe放到index.js同级目录，本程序并不会读取环境变量%path%中的可执行文件
## main.js是windows特供版的base file，为尊重原作者故不删除
## 本程序需要的文件如下，所有文件需要在同一文件夹里:
+ node.exe
+ ffmpeg.exe
+ cctv.worker.new.js
+ main.mjs
+ index.js
### 运行时执行node.exe index.js 程序会提示用户输入GUID值
### 本程序只支持下载2000.m3u8的流，450,850,4000这三个流可以用网上的油猴脚本
